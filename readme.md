# slack-spellfucker

just for fun, does not implement verification.. yet

![](http://i.imgur.com/xV9Vozp.png)

## setup

1. [create an app](https://api.slack.com/apps/new)
2. register a command to https://${YOUR_APP}/commands/spellfucker

## development

```
$ npm i
$ npm run dev
```

## production

```
$ PORT=3001 npm start
```
